# IoT Based Health Monitoring System Using Raspberry Pi

## Overview

This project is an IoT-based Health Monitoring System using Raspberry Pi. It monitors health parameters like heart rate, body temperature, ECG, and fall detection. The data can be viewed remotely through the internet.

## Key Features

* Heart rate monitoring
* ECG monitoring
* Temperature monitoring
* Fall detection
* Real-time monitoring
* Cloud data storage

## Tools and Technology

* Raspberry Pi 4
* Python
* Raspberry Pi OS
* DHT11 Sensor
* ECG Sensor
* ADXL335
* MCP3208 ADC

## Design Flow

* Connect sensors to Raspberry Pi
* Read sensor data
* Process the data
* Upload data to the cloud
* Display health information
